Challenge

I found that the options for styling HTML5 range inputs were inadequate. Since 2014, range inputs have become fairly powerful but integrating them is complicated.

Solution

Using LESS and Angular, I built this tool which you can use to visually style your sliders and then automatically generate CSS for your website.
