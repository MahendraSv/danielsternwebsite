Challenge

I wanted to create a library to reliably make creative sound and music with the Web Audio API.

Native audio is fast and effecient compared to formats like .mp3, and I wanted a way to take advantage of this.

Solution

I needed the library to be robust and efficient, since uses an API that's still being developed. So I documented it thoroughly, complete with graphics. The result is a multimedia experience.

Using Angular.js and LESS, I created numerous visualizers to help people imagine what they can do with the library.
